# 백엔드 및 AI 서버 시작 계획

## 🎯 문제 점유 및 해결 목표
- 로컬 개발 환경에서 백엔드(Spring Boot)와 AI 서버(FastAPI)가 정상적으로 동작하도록 실행합니다.
- 기존에 실행 중인 프로세스가 있다면 정리하여 포트 충돌을 방지합니다.

## 🛠 실행 단계 (Proposed Execution)

### 1. 포트 확인 및 기존 프로세스 정리
- **AI 서버 포트 (8000)**: `lsof -i :8000`으로 프로세스 확인 후 종료
- **백엔드 서버 포트 (8080)**: `lsof -i :8080`으로 프로세스 확인 후 종료

### 2. AI 서버 실행 (Python/FastAPI)
- **위치**: `ai-service/`
- **명령**: `python3 main.py` (또는 `python main.py`)
- **방법**: 백그라운드로 실행하거나 별도 터미널 세션 유지

### 3. 백엔드 서버 실행 (Java/Spring Boot)
- **위치**: `backend/`
- **명령**: `./gradlew bootRun`
- **방법**: 백그라운드로 실행

## 🚀 기대 효과
- 전체 시스템(FE 제외) 정상 기동 확인.
- API 통신 준비 완료.
